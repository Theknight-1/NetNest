## Getting Started

First, run the backend server:

```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```

Second, run the Frontend server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
